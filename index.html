Featured Work

File Hasher

File Hasher is a clean UI to hash files to help ensure the equality of two files. It is a RUST-WASM based implementation, and uses audited OSS as the basis for the cryptographic hashing. All hashing is done locally on the browser. No files or information is sent to any server.

Restructure-yt

Restructure-yt is a full service REST api and UI for downloading audio files from youtube videos. It is constructed with TypeScript, React, Node.js, Express, and the yt-dlp API.
